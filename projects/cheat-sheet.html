Key Array Methods:

  • .compact Returns the Array without element that equal nil
  • .delete_if { |object| ... } Deletes elements for which the block returns true
  • .each { |object| ... } Sends each element to the block
  • .join Returns a String with all elements separated by separator
  • .reverse Reverses the order of elements
  • .rindex Object object... Returns the index of the last occurrence of object

Key Hash Methods:

  • .include? .key? Returns true if the hash includes a value for key
  • .value? Returns true if the collection includes a key for value
  • .invert Returns a new Hash with keys and values inverted
  • .merge Returns a new Hash with entries from both Hashes
  • .select { |object| ... } Returns an Array with key-value pairs for which the block returns true
  • .to_a Returns an Array with nested key-value pairs

JAVASCRIPT

Conditional and Loop Syntax


The For Loop
Syntax:
  1. Statement 1 is executed before the loop (the code block) starts.
  2. Statement 2 defines the condition for running the loop (the code block).
  3. Statement 3 is executed each time after the loop (the code block) has been executed.

Example
for (i = 0; i < 5; i++) {
text += "The number is " + i + "
";
}

The While Loop
Syntax:
while (i < 10) {
text += "The number is " + i;
i++;
}
= The code in the loop will run, over and over again, as long as a variable (i) is less than 10.

COMPOSITE EXPRESSIONS

a[b]: Subscript, fetch the field named by b from value a.
a.x: Shorthand for a["x"].
a(b): Function call. Call the function value a with b as argument. Zero or more argument expressions can be given, separated by spaces. a(1, 2, 3, 4)
a.x(b): Method call. Call the function found in field x of value a, and pass a as the this argument.
